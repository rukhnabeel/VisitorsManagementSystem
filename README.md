# TripVenza Visitor System

A premium MERN stack visitor management system for TripVenza Holidays.

## Features
- **Premium UI**: Modern dark theme with glassmorphism and smooth animations.
- **Visitor Registration**: Easy-to-use form with real-time feedback.
- **Admin Dashboard**: Secure panel to manage visitors, search, and approve/reject requests.
- **Email Notifications**: Automatic emails to visitors upon approval or rejection.
- **QR Code**: Instant registration via QR code scanning in the waiting area.

## Tech Stack
- **Frontend**: React.js (Vite), Tailwind CSS, Framer Motion, Lucide Icons.
- **Backend**: Node.js, Express.js, MongoDB (Mongoose), Nodemailer.
- **QR Code**: qrcode.react.

## Setup Instructions

### 1. Prerequisites
- Node.js installed.
- MongoDB Atlas account (or local MongoDB).

### 2. Backend Setup
1. Navigate to `backend/`.
2. Copy `.env.example` to `.env`.
3. Fill in your `MONGO_URI`, `EMAIL_USER`, and `EMAIL_PASS` (Gmail App Password).
4. Run `npm install`.
5. Start server: `node server.js` (or `npm start` if script added).

### 3. Frontend Setup
1. Navigate to `frontend/`.
2. Run `npm install`.
3. Start development server: `npm run dev`.

### 4. Admin Credentials
- **Username**: `admin`
- **Password**: `tripvenza2026`

## Deployment
- **Frontend**: Deploy to Vercel/Netlify.
- **Backend**: Deploy to Render/Railway.
- **Database**: Use MongoDB Atlas.
